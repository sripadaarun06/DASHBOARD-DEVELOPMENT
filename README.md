- **Objective**: Visualize and analyze user activity across key touchpoints in an e-commerce platform.

- **Tools Used**: Power BI, Power Query (M), DAX

## Features

**Date Filtering** ‚Äì Dynamic filtering by event date

**Unique Visitors** ‚Äì Count of unique users by session

**Event Analysis** ‚Äì Breakdown of events (View, Cart, Purchase)

**Conversion Funnel** ‚Äì Visual funnel tracking user progression

**Top Products** ‚Äì Most viewed and most purchased items

**ETL** ‚Äì Cleaned timestamp conversion and error handling in Power Query
